Detailed Description

Status and error reporting types for Statefox.

Defines sf_status_t, the return type used by all Statefox API functions, along with sf_code_t for machine-readable error codes and sf_status_is_okay() for convenient success checks.

This header has no dependencies on the FSM itself and may be used independently in other projects.
